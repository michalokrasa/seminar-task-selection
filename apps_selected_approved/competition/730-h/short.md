Given n files and a subset of m files to delete, construct a pattern using only literal characters and `?` wildcards (each `?` matches any single character) that matches all files to delete and no others. A pattern matches a filename only if they have equal length. If multiple valid patterns exist, print any; if none exists, print "No". The algorithmic approach is constructive: first verify all target files share the same length (otherwise "No"), then build the pattern position by position — use the literal character if all targets agree at that position, otherwise `?` — and finally verify no non-target file of the same length matches the resulting pattern. The main challenge is this final verification step.
