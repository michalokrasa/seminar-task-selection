# Most Frequent Two-Gram

## Goal

Identify the most frequently occurring two-character substring (two-gram) in a given string of capital Latin letters.

## Rules

- A two-gram is a substring consisting of two consecutive characters.
- The string may contain overlapping two-grams.
- If multiple two-grams have the same maximum frequency, any one of them can be returned.

## Input / Output

- **Input**: 
  - An integer `n` (2 ≤ n ≤ 100) representing the length of the string.
  - A string `s` of length `n` consisting of capital Latin letters.
- **Output**: 
  - A string of exactly two capital Latin letters representing the most frequent two-gram.
