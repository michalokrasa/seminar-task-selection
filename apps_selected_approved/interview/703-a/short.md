Mishka and Chris play n rounds of dice; each round both roll a six-sided die and the player with the higher value wins that round (ties award neither player). After all rounds, the player with more round wins is declared the overall winner. If both players won the same number of rounds, print "Friendship is magic!^^". The algorithmic approach is straightforward simulation with per-round comparison and win counters; the main challenge is outputting the exact draw string without typos.
