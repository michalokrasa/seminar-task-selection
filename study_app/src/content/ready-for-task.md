## **You're ready to start**

The next step is the **first task**. You will be given a problem description and asked to write a prompt that would let an LLM solve it.

Take your time — there is no time limit, and you can save a draft at any point before submitting.

Your progress is saved automatically as you go — including an **autosave
every 30 seconds** while you're writing a prompt — so you can safely close
this tab and come back later; you'll resume exactly where you left off.

You will have access to the Gherkin intro and prompt guidelines at all times.

**_Remember not to use AI to write the prompt for you. The goal of this study is to understand how you approach writing prompts, so using an LLM to do it for you would defeat the purpose._**
