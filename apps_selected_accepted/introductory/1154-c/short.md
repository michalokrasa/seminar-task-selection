A cat follows a fixed weekly diet: fish on Mon/Thu/Sun, rabbit stew on Tue/Sat, and chicken on Wed/Fri. Given fixed supplies of each food type, choose the best day of the week to start a trip so the cat can eat for as many consecutive days as possible without running out of any food. The algorithmic approach is to try all 7 starting days and for each use integer division to count how many full weeks fit, then simulate the remaining partial week day by day. The main challenge is avoiding naive day-by-day simulation given supply values up to 7×10^8.
