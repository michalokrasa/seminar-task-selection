Given n coins each with an integer value, distribute all coins into pockets such that no two coins with the same value go into the same pocket. The goal is to use the minimum number of pockets possible. The answer equals the maximum frequency of any single coin value in the array. The algorithmic approach is simple frequency counting (e.g. with a Counter or dictionary); the main challenge is recognising that no simulation is needed — the answer follows directly from the maximum frequency.
