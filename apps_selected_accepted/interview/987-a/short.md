The Infinity Gauntlet holds six gems, each with a unique color: Power (purple), Time (green), Space (blue), Soul (orange), Reality (red), and Mind (yellow). Given the colors of the gems currently present, output the count and names of the gems that are absent. The names should be printed with the first letter capitalized. The algorithmic approach is a simple set difference using a fixed color-to-name lookup table; the main challenge is correctly implementing the mapping and ensuring the output format (capitalised names, correct count) matches the specification.
