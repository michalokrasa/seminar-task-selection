Santa suspects some pairs of keys were swapped on his keyboard, where each key is either in its original position or exchanged with exactly one other key. Given the intended string and the string that was actually typed, deduce which key pairs were swapped. Output the number of pairs followed by each pair, or -1 if the two strings are inconsistent with any valid swap configuration (e.g. a letter would need to map to two different letters). The algorithmic approach is to build a bidirectional character mapping between the typed and intended strings, checking consistency at each position; the main challenge is detecting contradictions where a character would need to map to two different targets, or two different characters map to the same target.
