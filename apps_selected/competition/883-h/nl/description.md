# Minimum Palindrome Parts

## Goal

Determine the minimum number of equal-length palindromes into which a given string can be rearranged and divided.

## Rules

- You can rearrange the characters of the string.
- Each part must be a palindrome.
- All parts must have the same length.

## Input / Output

- **Input**: 
  - An integer `n` (1 ≤ n ≤ 400,000) representing the length of the string.
  - A string `s` of length `n` consisting of lowercase and uppercase Latin letters and digits.
- **Output**: 
  - An integer `k` representing the minimum number of palindromes.
  - `k` palindromes of equal length, separated by spaces. The order of palindromes does not matter.
