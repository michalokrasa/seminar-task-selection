# Minimum Operations to Form a Perfect Square

## Goal

Determine the minimum number of digit deletions required to transform a given integer into a perfect square, or determine if it is impossible.

## Rules

- You can delete any digit from the integer.
- The resulting number must be a perfect square.
- The number must remain a positive integer without leading zeros.

## Input / Output

- **Input**: A single integer `n` (1 ≤ n ≤ 2 × 10^9) given without leading zeroes.
- **Output**: The minimum number of deletions required to form a perfect square, or `-1` if impossible.
