# Maximum Days of Feeding

## Goal

Determine the maximum number of consecutive days Polycarp's cat can be fed without running out of food, starting from an optimal day of the week.

## Rules

- The cat eats:
  - Fish food on Mondays, Thursdays, and Sundays.
  - Rabbit stew on Tuesdays and Saturdays.
  - Chicken steak on Wednesdays and Fridays.
- Polycarp can choose any day of the week to start the trip.

## Input / Output

- **Input**: Three integers `a`, `b`, and `c` representing the number of rations of fish food, rabbit stew, and chicken steak, respectively. Constraints: \(1 \le a, b, c \le 7 \times 10^8\).
- **Output**: A single integer representing the maximum number of days the cat can be fed consecutively.
