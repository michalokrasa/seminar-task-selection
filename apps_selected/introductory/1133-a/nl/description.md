# Midpoint of Contest Time

## Goal

Calculate the midpoint time of a contest given its start and end times.

## Rules

- The contest starts and ends on the same day.
- The contest duration is an even number of minutes.
- The contest lasts at least two minutes.

## Input / Output

- **Input**: 
  - Two lines, each containing a time in the format `hh:mm`.
  - The first line is the start time (`h1:m1`).
  - The second line is the end time (`h2:m2`).
  - Constraints: `0 ≤ h1, h2 ≤ 23` and `0 ≤ m1, m2 ≤ 59`.

- **Output**: 
  - A single line with the midpoint time in the format `hh:mm`, with each number as exactly two digits.
