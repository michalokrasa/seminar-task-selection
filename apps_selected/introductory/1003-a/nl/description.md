# Minimum Pockets

## Goal

Determine the minimum number of pockets needed to distribute coins such that no two coins with the same value are in the same pocket.

## Rules

- Coins with the same value cannot be placed in the same pocket.
- The goal is to minimize the number of pockets used.

## Input / Output

- **Input**: 
  - The first line contains an integer `n` (1 ≤ n ≤ 100), the number of coins.
  - The second line contains `n` integers `a_1, a_2, ..., a_n` (1 ≤ a_i ≤ 100), representing the values of the coins.
- **Output**: 
  - A single integer representing the minimum number of pockets required.
