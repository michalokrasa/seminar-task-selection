A robot on a number line starts at 0 and receives n move commands; each command at time t_i tells it to go to point x_i at speed 1, but the robot ignores new commands while still moving. A command is "successful" if the robot visits its target point x_i at any time in [t_i, t_{i+1}]. Given the full command sequence, count how many commands are successful. The solution requires simulating the robot's position and reachable range within each time interval between consecutive commands.
