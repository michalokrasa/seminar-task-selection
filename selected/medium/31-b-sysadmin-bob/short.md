A string of email addresses has had all commas removed; recover any valid comma-separated list of email addresses of the form A@B. Each address must have exactly one '@', with non-empty local and domain parts consisting of lowercase letters/digits. The problem is to find a valid segmentation of the string, or report "No solution" if none exists. Since each address must contain exactly one '@', the number of '@' characters in the input determines how many addresses there are.
