Santa typed a string t intending to type string s, but some pairs of keys were swapped on his keyboard. Each key is either in its correct position or swapped with exactly one other key. Determine the set of key swaps that explains the discrepancy, or output -1 if no consistent set of at most 13 swaps exists. The solution builds a character mapping from s to t and checks for contradictions (a character mapping to two different characters, or a non-symmetric mapping).
