# Asterisks

## Goal

Print a triangular pattern of asterisks with `n` rows.

## Rules

- Print exactly `n` lines.
- Line `i` (1-based) consists of exactly `i` asterisk characters `*`.

## Input / Output

- **Input**: a single integer `n` (1 ≤ n ≤ 50).
- **Output**: `n` lines, where line `i` contains `i` asterisks.
