# Spy Detected!

## Goal

Find the 1-based index of the unique element in an array where all other elements are identical.

## Rules

- The array has at least 3 elements and at most 100.
- All elements except exactly one share the same value.
- Return the 1-based index of the element that differs from the rest.

## Input / Output

- **Input**: first line is `t` (1 ≤ t ≤ 100, number of test cases). For each test case: first line is `n` (3 ≤ n ≤ 100), second line contains `n` space-separated integers (1 ≤ a_i ≤ 100).
- **Output**: for each test case, a single integer — the 1-based index of the unique element.
